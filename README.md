# Project Summary - DIY1: Antigravity and GitHub

## Overview
Today, I developed and deployed a modern personal webpage for **李亦騰 (Li Yiteng)**. The project involved creating a stylish frontend using HTML/CSS/JavaScript and setting up a version control workflow with GitHub.

* **Live Demo:** https://eeebooz-gif.github.io/0224DIY1/
*(Note: Please ensure GitHub Pages is enabled in Settings to view the live demo)*

## Completed Tasks

### 1. Web Development
* **Main Interface (index.html):** Built a single-page layout featuring a dynamic greeting and a real-time clock display.
* **Visual Design:** Implemented a **"Glassmorphism"** design style with frosted-glass effects, a deep purple animated gradient background, and a shimmering text effect for the name display.
* **Dynamic Features:** Added JavaScript to handle real-time clock (hours, minutes, seconds) and full date updates.

### 2. Git & GitHub Integration
* **Repository Setup:** Initialized a new Git repository in the `L1` directory and linked it to the remote GitHub repository.
* **Identity Configuration:** Configured local Git user settings (`user.name` and `user.email`) to correctly match my identity (**李亦騰** / **eeebooz@gmail.com**).
* **Troubleshooting:** Successfully resolved "Permission denied (403)" and "Repository not found" errors by updating the remote URL and authenticating via browser.
* **Deployment:** Pushed the complete codebase to the remote repository: https://github.com/eeebooz-gif/0224DIY1.git

## File Structure
* `index.html`: Contains the core structure, CSS styling (Glassmorphism), and JavaScript logic.
* `README.md`: Project documentation and summary.
